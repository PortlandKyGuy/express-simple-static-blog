Getting Started with express-simple-static-blog

Setting up a blog with express-simple-static-blog is incredibly easy. Here's how:

Installation

npm install express-simple-static-blog

Basic Setup

const express = require('express');
const expressStaticBlog = require('express-simple-static-blog');

const app = express();

// Set up view engine
app.set('view engine', 'ejs');
app.set('views', './views');

// Initialize blog
const blog = expressStaticBlog({
    blogsDir: './views/blogs',
    routePrefix: '/blog'
});

// Mount blog router
app.use(blog.router());

app.listen(3000);

Creating Blog Posts

Just create HTML files in your blogs directory with the proper metadata format:

<!-- [date: 2025-01-19] -->
<!-- [desc: Your post description] -->

That's it! Your blog is ready to go.
